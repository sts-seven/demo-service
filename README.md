# demo-app
```
提交代码
1、添加跟踪文件（修改新增的文件）
   git add .   或者  git add -A 
2、提交代码并写注释
   git commit -m "提交的注释"
   如果使用了 git commit 命令会进入编辑备注页面，可以按Insert 键或者 I 键插入文本，按ESC返回，输入:wq 结束输入。
3、将本地仓库推送到远程服务器。
   git push origin master

更新代码
1、git pull orgin master 将远程服务器的代码更新下来

GitHub可视化工具：GitHubDesktopSetup.exe
开发工具：visual studio code
HTTP工具：Postman
后端数据库：mongoDB 服务安装，可视化工具studio 3T
前端环境安装：nodejs、angular cli 、ionic 
技术：typescript  ionic3
```
